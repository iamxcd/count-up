# count-up
> 数字日历翻动效果

## 安装
```
npm i @songbai_x/count-up
```
## 使用
```vue
<template>
<CountUp :num="num">

</template>

import CountUp from "@songbai_x/count-up"

componsents:{
    CountUp
},
data(){
    return {
        num:2019
    }
}
```
